    Introducción a PHP

    1. ¿Qué es PHP?

    PHP (PHP: Hypertext Preprocessor) es un lenguaje de programación de código abierto especialmente diseñado para el desarrollo web del lado del servidor.

    • Es ejecutado en el servidor, generando HTML que se envía al cliente
    • Puede integrarse fácilmente con HTML
    • Es compatible con múltiples bases de datos (MySQL, PostgreSQL, etc.)
    • Es multiplataforma (Windows, Linux, macOS)
    • Es gratuito y de código abierto

    1.1. ¿Para qué se usa PHP?

    • Crear páginas web dinámicas
    • Recoger datos de formularios
    • Gestionar bases de datos
    • Crear sistemas de autenticación
    • Generar contenido personalizado

    2. Instalación de PHP

    2.1. Opciones de instalación

    Para desarrollar con PHP necesitas un servidor web con PHP instalado. Las opciones más comunes son:

    • XAMPP: Paquete todo-en-uno (Apache + MySQL + PHP)
    • WAMP: Para Windows (Windows + Apache + MySQL + PHP)
    • MAMP: Para macOS
    • Servidor PHP integrado: Desde PHP 5.4+ incluye un servidor de desarrollo

    2.2. XAMPP

    XAMPP crea una carpeta htdocs donde debes colocar tus archivos PHP. El servidor Apache lee y ejecuta los archivos desde esta ubicación.

    • Windows: C:\xampp\htdocs\
    • macOS: /Applications/XAMPP/htdocs/

    Para acceder a tus archivos desde el navegador: http://localhost/index.php

    3. Sintaxis básica de PHP

    Los scripts PHP comienzan con <?php y terminan con ?> 

    
<?php
// Este es un script PHP básico
echo "¡Hola, Mundo!";
?>

    3.1. Integración con HTML

    
<!DOCTYPE html>
<html>
<head>
    <title>Mi página PHP</title>
</head>
<body>
    <h1><?php echo "Bienvenido a PHP"; ?></h1>
    <p>Hoy es <?php echo date("d/m/Y"); ?></p>
</body>
</html>

    3.2. Reglas importantes

    • Los archivos PHP deben tener extensión .php
    • Las instrucciones terminan con punto y coma ;
    • PHP es case-sensitive para variables, pero no para palabras clave

    4. Comentarios en PHP

    Los comentarios se usan para documentar el código y no se ejecutan.

    
<?php
// Este es un comentario de una línea

# También se puede usar # para comentarios de una línea

/*
  Este es un comentario
  de múltiples líneas
  que puede ocupar varias líneas
*/

echo "Los comentarios no se muestran"; // Comentario al final de la línea
?>

    5. Variables en PHP

    Las variables en PHP comienzan con el símbolo $ seguido del nombre de la variable. 

    
<?php
// Declaración de variables
$nombre = "Juan";           // String (cadena de texto)
$edad = 25;                 // Integer (número entero)
$precio = 19.99;            // Float (número decimal)
$activo = true;             // Boolean (verdadero/falso)

// Las variables son case-sensitive
$color = "rojo";
$Color = "azul";            // Variable diferente
echo $color;                // Imprime: rojo
?>

    5.1. Reglas para nombres de variables

    • Deben comenzar con $
    • Después de $ debe ir una letra o guión bajo
    • Solo pueden contener letras, números y guiones bajos
    • Son case-sensitive ($nombre ≠ $Nombre)

    5.2. Variables dinámicas

    
<?php
$nombre_variable = "edad";
$$nombre_variable = 30;     // Crea $edad = 30

echo $edad;                 // Imprime: 30
?>

    6. Echo y Print

    echo y print se usan para mostrar contenido en pantalla.

    6.1. Echo

    
<?php
// Echo puede mostrar múltiples valores separados por comas
echo "Hola", " ", "Mundo";

// Echo con variables
$nombre = "María";
echo "Hola, $nombre";       // Interpolación de variables

// Echo con concatenación
echo "Hola, " . $nombre;    // Operador de concatenación .
?>

    6.2. Print

    
<?php
// Print solo puede mostrar un valor
print "Hola Mundo";

// Print devuelve 1, por lo que puede usarse en expresiones
$resultado = print "Texto";  // $resultado = 1
?>

    6.3. Diferencias entre echo y print

    • echo: Más rápido, puede recibir múltiples parámetros, no devuelve valor
    • print: Más lento, un solo parámetro, devuelve 1

    7. Tipos de datos en PHP

    PHP soporta varios tipos de datos:

    
<?php
// 1. String (cadena de texto)
$texto = "Hola Mundo";
$texto2 = 'También con comillas simples';

// 2. Integer (número entero)
$numero = 42;
$negativo = -15;

// 3. Float/Double (número decimal)
$decimal = 3.14;
$cientifico = 1.5e3;        // 1500

// 4. Boolean (verdadero o falso)
$verdadero = true;
$falso = false;

// 5. Array (arreglo)
$colores = array("rojo", "verde", "azul");
$numeros = [1, 2, 3, 4, 5]; // Sintaxis corta

// 6. Object (objeto)
class Persona {
    public $nombre;
}
$persona = new Persona();
$persona->nombre = "Juan";

// 7. NULL (valor nulo)
$vacio = null;

// Verificar tipo de dato
echo gettype($numero);      // Imprime: integer
var_dump($colores);         // Muestra información detallada
?>

    8. Strings (Cadenas de texto)

    8.1. Operaciones con strings

    
<?php
$nombre = "Juan";
$apellido = "García";

// Concatenación
$nombre_completo = $nombre . " " . $apellido;
echo $nombre_completo;      // Imprime: Juan García

// Longitud de una cadena
echo strlen($nombre_completo);  // Imprime: 12

// Contar palabras
echo str_word_count($nombre_completo);  // Imprime: 2

// Invertir cadena
echo strrev($nombre);       // Imprime: nauJ

// Buscar texto
echo strpos("Hola Mundo", "Mundo");  // Imprime: 5 (posición)
?>

    8.2. Funciones útiles de strings

    
<?php
$texto = "Hola Mundo PHP";

// Convertir a mayúsculas y minúsculas
echo strtoupper($texto);    // HOLA MUNDO PHP
echo strtolower($texto);    // hola mundo php

// Reemplazar texto
echo str_replace("Mundo", "Amigo", $texto);  // Hola Amigo PHP

// Extraer subcadena
echo substr($texto, 0, 4);  // Hola

// Dividir cadena en array
$palabras = explode(" ", $texto);  // ["Hola", "Mundo", "PHP"]

// Unir array en cadena
$nueva_cadena = implode("-", $palabras);  // Hola-Mundo-PHP
?>

    9. Numbers (Números)

    9.1. Integers (Enteros)

    
<?php
$entero = 42;
$negativo = -100;
$hexadecimal = 0xFF;        // 255 en decimal

// Verificar si es entero
var_dump(is_int($entero));  // bool(true)

// Valor máximo y mínimo
echo PHP_INT_MAX;           // 9223372036854775807 (en 64 bits)
echo PHP_INT_MIN;           // -9223372036854775808
?>

    9.2. Floats (Decimales)

    
<?php
$decimal = 3.14;
$cientifico = 2.5e3;        // 2500

// Verificar si es float
var_dump(is_float($decimal));  // bool(true)

// Verificar si es numérico
var_dump(is_numeric("123"));   // bool(true)
var_dump(is_numeric("abc"));   // bool(false)
?>

    9.3. Operaciones numéricas

    
<?php
$a = 10;
$b = 3;

echo $a + $b;               // Suma: 13
echo $a - $b;               // Resta: 7
echo $a * $b;               // Multiplicación: 30
echo $a / $b;               // División: 3.333...
echo $a % $b;               // Módulo (resto): 1
echo $a ** $b;              // Potencia: 1000
?>

    10. Casting (Conversión de tipos)

    El casting permite convertir una variable de un tipo a otro. 

    
<?php
// Convertir a string
$numero = 123;
$texto = (string)$numero;
echo gettype($texto);       // string

// Convertir a integer
$decimal = 3.14;
$entero = (int)$decimal;
echo $entero;               // 3 (se trunca, no redondea)

// Convertir a float
$numero = 42;
$decimal = (float)$numero;
echo $decimal;              // 42.0

// Convertir a boolean
$valor = 1;
$booleano = (bool)$valor;
var_dump($booleano);        // bool(true)

// Valores que se convierten a false
var_dump((bool)0);          // false
var_dump((bool)"");         // false
var_dump((bool)null);       // false
var_dump((bool)[]);         // false

// Convertir a array
$numero = 10;
$array = (array)$numero;
print_r($array);            // Array([0] => 10)
?>

    11. Funciones matemáticas

    
<?php
// Funciones básicas
echo abs(-5);               // Valor absoluto: 5
echo ceil(3.2);             // Redondear hacia arriba: 4
echo floor(3.8);            // Redondear hacia abajo: 3
echo round(3.5);            // Redondear: 4

// Potencias y raíces
echo pow(2, 3);             // Potencia: 8 (2^3)
echo sqrt(16);              // Raíz cuadrada: 4

// Máximo y mínimo
echo max(1, 5, 3, 9, 2);    // Valor máximo: 9
echo min(1, 5, 3, 9, 2);    // Valor mínimo: 1

// Números aleatorios
echo rand();                // Número aleatorio
echo rand(1, 100);          // Entre 1 y 100

// Constantes matemáticas
echo M_PI;                  // 3.14159265358979
echo M_E;                   // 2.7182818284590 (número e)

// Funciones trigonométricas
echo sin(M_PI / 2);         // Seno: 1
echo cos(0);                // Coseno: 1
echo tan(M_PI / 4);         // Tangente: 1
?>

    12. Constantes

    Las constantes son como variables pero su valor no puede cambiar una vez definido.

    12.1. Definir constantes

    
<?php
// Método 1: usando define()
define("SITIO", "Mi Web");
define("VERSION", 1.5);
define("PI", 3.14159);

echo SITIO;                 // Mi Web

// Método 2: usando const (desde PHP 5.3)
const PAIS = "España";
const EDAD_MINIMA = 18;

echo PAIS;                  // España

// Las constantes son globales
function mostrarVersion() {
    echo VERSION;           // Se puede acceder desde cualquier lugar
}
?>

    12.2. Constantes predefinidas

    
<?php
// Constantes útiles de PHP
echo PHP_VERSION;           // Versión de PHP
echo PHP_OS;                // Sistema operativo
echo PHP_INT_MAX;           // Valor entero máximo

// Verificar si existe una constante
if (defined("SITIO")) {
    echo "La constante SITIO existe";
}
?>

    13. Constantes mágicas

    PHP tiene constantes especiales que cambian dependiendo del contexto. 

    
<?php
// __LINE__ - Número de línea actual
echo "Esta es la línea: " . __LINE__;

// __FILE__ - Ruta completa del archivo
echo "Archivo: " . __FILE__;

// __DIR__ - Directorio del archivo
echo "Directorio: " . __DIR__;

// __FUNCTION__ - Nombre de la función actual
function miFuncion() {
    echo "Función: " . __FUNCTION__;
}

// __CLASS__ - Nombre de la clase actual
class MiClase {
    public function mostrar() {
        echo "Clase: " . __CLASS__;
    }
}

// __METHOD__ - Nombre del método actual
class Ejemplo {
    public function metodo() {
        echo "Método: " . __METHOD__;  // Ejemplo::metodo
    }
}

// __NAMESPACE__ - Espacio de nombres actual
namespace MiApp;
echo "Namespace: " . __NAMESPACE__;
?>

    14. Operadores en PHP

    14.1. Operadores aritméticos

    
<?php
$a = 10;
$b = 3;

echo $a + $b;               // Suma: 13
echo $a - $b;               // Resta: 7
echo $a * $b;               // Multiplicación: 30
echo $a / $b;               // División: 3.333...
echo $a % $b;               // Módulo: 1
echo $a ** $b;              // Potencia: 1000
?>

    14.2. Operadores de asignación

    
<?php
$x = 10;                    // Asignación simple
$x += 5;                    // $x = $x + 5 -> 15
$x -= 3;                    // $x = $x - 3 -> 12
$x *= 2;                    // $x = $x * 2 -> 24
$x /= 4;                    // $x = $x / 4 -> 6
$x %= 5;                    // $x = $x % 5 -> 1

$texto = "Hola";
$texto .= " Mundo";         // Concatenación: "Hola Mundo"
?>

    14.3. Operadores de comparación

    
<?php
$a = 5;
$b = "5";

var_dump($a == $b);         // true (igual en valor)
var_dump($a === $b);        // false (idéntico: valor y tipo)
var_dump($a != $b);         // false (diferente)
var_dump($a !== $b);        // true (no idéntico)
var_dump($a > 3);           // true (mayor que)
var_dump($a < 10);          // true (menor que)
var_dump($a >= 5);          // true (mayor o igual)
var_dump($a <= 5);          // true (menor o igual)

// Operador spaceship (PHP 7+)
echo 5 <=> 3;               // 1 (5 mayor que 3)
echo 3 <=> 5;               // -1 (3 menor que 5)
echo 5 <=> 5;               // 0 (son iguales)
?>

    14.4. Operadores lógicos

    
<?php
$x = true;
$y = false;

var_dump($x && $y);         // AND: false
var_dump($x || $y);         // OR: true
var_dump(!$x);              // NOT: false
var_dump($x and $y);        // AND (menor precedencia): false
var_dump($x or $y);         // OR (menor precedencia): true
var_dump($x xor $y);        // XOR (exclusivo): true
?>

    14.5. Operadores de incremento/decremento

    
<?php
$contador = 5;

echo ++$contador;           // Pre-incremento: 6 (incrementa y luego usa)
echo $contador++;           // Post-incremento: 6 (usa y luego incrementa)
echo $contador;             // 7

echo --$contador;           // Pre-decremento: 6
echo $contador--;           // Post-decremento: 6
echo $contador;             // 5
?>

    15. Condicionales If...Else...Elseif

    15.1. If simple

    
<?php
$edad = 18;

if ($edad >= 18) {
    echo "Eres mayor de edad";
}
?>

    15.2. If...Else

    
<?php
$temperatura = 15;

if ($temperatura > 25) {
    echo "Hace calor";
} else {
    echo "Hace frío";
}
?>

    15.3. If...Elseif...Else

    
<?php
$nota = 7;

if ($nota >= 9) {
    echo "Sobresaliente";
} elseif ($nota >= 7) {
    echo "Notable";
} elseif ($nota >= 5) {
    echo "Aprobado";
} else {
    echo "Suspenso";
}
?>

    15.4. Operador ternario

    
<?php
$edad = 20;

// Sintaxis: condición ? valor_si_true : valor_si_false
$resultado = ($edad >= 18) ? "Mayor" : "Menor";
echo $resultado;            // Mayor

// Operador null coalescing (PHP 7+)
$nombre = $_GET['nombre'] ?? "Anónimo";  // Si no existe, usa "Anónimo"
?>

    16. Switch

    La estructura switch evalúa una expresión y ejecuta código según diferentes casos. 

    
<?php
$dia = 3;

switch ($dia) {
    case 1:
        echo "Lunes";
        break;
    case 2:
        echo "Martes";
        break;
    case 3:
        echo "Miércoles";
        break;
    case 4:
        echo "Jueves";
        break;
    case 5:
        echo "Viernes";
        break;
    case 6:
    case 7:
        echo "Fin de semana";
        break;
    default:
        echo "Día inválido";
}
?>

    16.1. Switch con múltiples valores

    
<?php
$mes = "enero";

switch ($mes) {
    case "diciembre":
    case "enero":
    case "febrero":
        echo "Invierno";
        break;
    case "marzo":
    case "abril":
    case "mayo":
        echo "Primavera";
        break;
    case "junio":
    case "julio":
    case "agosto":
        echo "Verano";
        break;
    case "septiembre":
    case "octubre":
    case "noviembre":
        echo "Otoño";
        break;
    default:
        echo "Mes no válido";
}
?>

    17. Bucles (Loops)

    17.1. While

    
<?php
// While: se ejecuta mientras la condición sea verdadera
$contador = 1;

while ($contador <= 5) {
    echo "Número: $contador
    ";
    $contador++;
}
// Imprime: 1, 2, 3, 4, 5
?>

    17.2. Do...While

    
<?php
// Do-While: se ejecuta al menos una vez
$numero = 10;

do {
    echo "Número: $numero
    ";
    $numero++;
} while ($numero < 10);         // La condición es falsa, pero se ejecuta una vez
?>

    17.3. For

    
<?php
// For: bucle con inicialización, condición e incremento
for ($i = 1; $i <= 5; $i++) {
    echo "Iteración: $i
    ";
}

// For con saltos
for ($i = 0; $i <= 10; $i += 2) {
    echo $i . " ";              // 0 2 4 6 8 10
}
?>

    17.4. Foreach (para arrays)

    
<?php
// Foreach: recorre arrays
$colores = ["rojo", "verde", "azul"];

foreach ($colores as $color) {
    echo "Color: $color
    ";
}

// Foreach con índice y valor
$edades = ["Juan" => 25, "María" => 30, "Pedro" => 28];

foreach ($edades as $nombre => $edad) {
    echo "$nombre tiene $edad años
    ";
}
?>

    17.5. Break y Continue

    
<?php
// Break: sale del bucle
for ($i = 1; $i <= 10; $i++) {
    if ($i == 5) {
        break;                  // Sale del bucle cuando i = 5
    }
    echo $i . " ";              // 1 2 3 4
}

echo "
    ";

// Continue: salta a la siguiente iteración
for ($i = 1; $i <= 10; $i++) {
    if ($i % 2 == 0) {
        continue;               // Salta los números pares
    }
    echo $i . " ";              // 1 3 5 7 9
}
?>

    17.6. Bucles anidados

    
<?php
// Tabla de multiplicar
for ($i = 1; $i <= 3; $i++) {
    for ($j = 1; $j <= 3; $j++) {
        $resultado = $i * $j;
        echo "$i x $j = $resultado
    ";
    }
    echo "---
    ";
}
?>

    Resumen de PHP básico

    • PHP es un lenguaje del lado del servidor para desarrollo web
    • Las variables comienzan con $ y son case-sensitive
    • Soporta múltiples tipos de datos: string, int, float, bool, array, object, null
    • Usa echo o print para mostrar contenido
    • Incluye operadores aritméticos, lógicos y de comparación
    • Las estructuras de control incluyen if, switch
    • Los bucles disponibles son: while, do-while, for, foreach
    • Las constantes se definen con define() o const
