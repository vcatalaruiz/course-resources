    Arrays en PHP

    1. ¿Qué son los arrays?

    Los arrays son estructuras de datos fundamentales que permiten almacenar múltiples valores en una sola variable. En PHP, los arrays son muy flexibles y pueden contener diferentes tipos de datos.

    Un array es una colección de elementos que se almacenan bajo un único nombre de variable. Cada elemento tiene una posición (índice) o una clave (key) que lo identifica.

    1.1. Tipos de arrays en PHP

    • Indexed Arrays: Utilizan índices numéricos (0, 1, 2, ...)
    • Associative Arrays: Utilizan claves de cadena (nombres personalizados)
    • Multidimensional Arrays: Arrays que contienen otros arrays

    1.2. Ventajas de usar arrays

    • Almacenar múltiples valores relacionados
    • Facilitan la iteración sobre colecciones de datos
    • Permiten estructurar datos complejos
    • Reducen la necesidad de crear múltiples variables

    2. Arrays indexados

    Un array indexado es un array que utiliza índices numéricos (comenzando desde 0) para acceder a cada elemento. Cada posición del array está identificada por un número automático que comienza en 0 para el primer elemento, 1 para el segundo, y así sucesivamente.

    2.1. Crear un array indexado

    
<?php
// Método 1: Usando array()
$colores = array("Rojo", "Verde", "Azul");

// Método 2: Usando [] (sintaxis corta recomendada)
$numeros = [10, 20, 30, 40, 50];

// Método 3: Asignando elementos uno a uno
$frutas = [];
$frutas[0] = "Manzana";
$frutas[1] = "Plátano";
$frutas[2] = "Naranja";

echo $colores[0];  // Imprime: Rojo
echo $numeros[2];  // Imprime: 30
?>

    2.2. Obtener información del array

    
<?php
$nombres = ["Ana", "Bruno", "Carlos", "Diana"];

// count() - Número de elementos
echo count($nombres);  // Imprime: 4

// sizeof() - Alias de count()
echo sizeof($nombres); // Imprime: 4

// Verificar si existe un índice
if (isset($nombres[1])) {
    echo $nombres[1];  // Imprime: Bruno
}
?>

    3. Arrays asociativos

    Un array asociativo es un array que utiliza claves de texto personalizadas (strings) en lugar de índices numéricos para acceder a cada elemento. Esto permite usar nombres descriptivos como claves, haciendo el código más legible y organizado. Cada elemento se asocia a una clave única mediante la estructura "clave" => valor.

    3.1. Crear un array asociativo

    
<?php
// Método 1: Usando array()
$persona = array(
    "nombre" => "Juan",
    "edad" => 30,
    "ciudad" => "Madrid"
);

// Método 2: Usando [] (más moderno)
$coche = [
    "marca" => "Toyota",
    "modelo" => "Corolla",
    "año" => 2023,
    "precio" => 25000
];

// Acceder a los valores
echo $persona["nombre"];  // Imprime: Juan
echo $coche["marca"];     // Imprime: Toyota
?>

    3.2. Verificar claves y valores

    
<?php
$usuario = ["nombre" => "María", "email" => "maria@ejemplo.com"];

// array_key_exists() - Verificar si existe una clave
if (array_key_exists("nombre", $usuario)) {
    echo "La clave 'nombre' existe";
}

// in_array() - Verificar si existe un valor
if (in_array("maria@ejemplo.com", $usuario)) {
    echo "El email existe en el array";
}

// isset() - Alternativa para verificar
if (isset($usuario["nombre"])) {
    echo $usuario["nombre"];
}
?>

    4. Modificar arrays

    4.1. Agregar elementos

    
<?php
$lista = ["primero", "segundo"];

// Agregar al final (array indexado)
$lista[] = "tercero";  // Índice 2
$lista[] = "cuarto";   // Índice 3

// Usar array_push() para agregar múltiples elementos
array_push($lista, "quinto", "sexto");

// Agregar a un array asociativo
$datos = ["nombre" => "Juan"];
$datos["edad"] = 30;
$datos["ciudad"] = "Barcelona";

print_r($lista);   // Muestra toda la estructura
print_r($datos);
?>

    4.2. Eliminar elementos

    
<?php
$colores = ["Rojo", "Verde", "Azul", "Amarillo"];

// array_pop() - Eliminar último elemento
$ultimo = array_pop($colores);
echo $ultimo;  // Imprime: Amarillo

// array_shift() - Eliminar primer elemento
$primero = array_shift($colores);
echo $primero; // Imprime: Rojo

// unset() - Eliminar un elemento específico
unset($colores[0]);

// array_splice() - Eliminar y reemplazar
$numeros = [1, 2, 3, 4, 5];
array_splice($numeros, 1, 2);  // Elimina 2 elementos desde índice 1
// $numeros ahora es [1, 4, 5]
?>

    4.3. Modificar elementos

    
<?php
$precios = ["manzana" => 1.50, "platano" => 0.80, "naranja" => 1.20];

// Modificar un valor
$precios["manzana"] = 1.75;

// Cambiar una clave
$precios["platano"] = 2.50;

// array_replace() - Reemplazar múltiples valores
$nuevosPrecios = ["manzana" => 2.00, "uva" => 3.00];
$precios = array_replace($precios, $nuevosPrecios);
?>

    5. Iterar a través de arrays

    5.1. Usar foreach()

    
<?php
$frutas = ["Manzana", "Plátano", "Naranja"];

// Iterar sin índice
foreach ($frutas as $fruta) {
    echo $fruta . "<br>";
}

// Iterar con índice
foreach ($frutas as $indice => $fruta) {
    echo "$indice: $fruta<br>";
}

// Array asociativo
$persona = ["nombre" => "Carlos", "edad" => 25, "ciudad" => "Valencia"];

foreach ($persona as $clave => $valor) {
    echo "$clave: $valor<br>";
}
?>

    5.2. Usar bucles for

    
<?php
$numeros = [10, 20, 30, 40, 50];

// Bucle for tradicional
for ($i = 0; $i < count($numeros); $i++) {
    echo $numeros[$i] . "<br>";
}

// Bucle while
$j = 0;
while ($j < count($numeros)) {
    echo $numeros[$j] . "<br>";
    $j++;
}
?>

    6. Funciones útiles de arrays

    6.1. Búsqueda y ordenamiento

    
<?php
$numeros = [5, 2, 8, 1, 9, 3];

// array_search() - Buscar un valor
$posicion = array_search(8, $numeros);
echo $posicion;  // Imprime: 2

// in_array() - Verificar si existe
if (in_array(5, $numeros)) {
    echo "5 existe en el array";
}

// sort() - Ordenar ascendente
sort($numeros);        // [1, 2, 3, 5, 8, 9]

// rsort() - Ordenar descendente
rsort($numeros);       // [9, 8, 5, 3, 2, 1]

// asort() - Ordenar manteniendo claves
$datos = ["c" => 3, "a" => 1, "b" => 2];
asort($datos);         // ["a" => 1, "b" => 2, "c" => 3]
?>

    6.2. Transformación de arrays

    
<?php
$nombres = ["Ana", "Bruno", "Carlos"];
$edades = [25, 30, 28];

// array_combine() - Combinar arrays
$personas = array_combine($nombres, $edades);
// ["Ana" => 25, "Bruno" => 30, "Carlos" => 28]

// array_flip() - Intercambiar claves y valores
$datos = ["nombre" => "Juan", "edad" => 30];
$invertido = array_flip($datos);
// [30 => "edad", "Juan" => "nombre"]

// array_reverse() - Invertir orden
$lista = [1, 2, 3, 4, 5];
$invertida = array_reverse($lista);
// [5, 4, 3, 2, 1]

// array_merge() - Fusionar arrays
$array1 = ["a" => 1, "b" => 2];
$array2 = ["c" => 3, "d" => 4];
$fusionado = array_merge($array1, $array2);
?>

    6.3. Funciones de cantidad y suma

    
<?php
$numeros = [10, 20, 30, 40, 50];

// count() / sizeof() - Contar elementos
echo count($numeros);  // Imprime: 5

// array_sum() - Sumar elementos
echo array_sum($numeros);  // Imprime: 150

// array_product() - Multiplicar elementos
echo array_product([2, 3, 4]);  // Imprime: 24

// array_unique() - Eliminar duplicados
$datos = [1, 2, 2, 3, 3, 3, 4];
$unicos = array_unique($datos);
// [1, 2, 3, 4]
?>

    7. Arrays multidimensionales

    Un array multidimensional es un array que contiene otros arrays como elementos. Permite estructurar datos complejos en múltiples niveles, como tablas o estructuras jerárquicas. Los arrays multidimensionales más comunes son los arrays 2D (bidimensionales), que funcionan como matrices con filas y columnas, pero también pueden tener más niveles de profundidad.

    7.1. Arrays indexados multidimensionales

    
<?php
// Array 2D - Tabla
$estudiantes = [
    ["Ana", 25, "Madrid"],
    ["Bruno", 22, "Barcelona"],
    ["Carlos", 23, "Valencia"],
    ["Diana", 24, "Bilbao"]
];

// Acceder a elementos
echo $estudiantes[0][0];     // Imprime: Ana
echo $estudiantes[1][2];     // Imprime: Barcelona
echo $estudiantes[2][1];     // Imprime: 23

// Iterar
foreach ($estudiantes as $fila) {
    echo "Nombre: " . $fila[0] . "<br>";
}
?>

    7.2. Arrays asociativos multidimensionales

    
<?php
// Lista de personas con propiedades
$personas = [
    [
        "nombre" => "Elena",
        "edad" => 28,
        "ciudad" => "Madrid",
        "email" => "elena@ejemplo.com"
    ],
    [
        "nombre" => "Fernando",
        "edad" => 32,
        "ciudad" => "Barcelona",
        "email" => "fernando@ejemplo.com"
    ],
    [
        "nombre" => "Gabriela",
        "edad" => 26,
        "ciudad" => "Sevilla",
        "email" => "gabriela@ejemplo.com"
    ]
];

// Acceder
echo $personas[0]["nombre"];  // Imprime: Elena
echo $personas[1]["email"];   // Imprime: fernando@ejemplo.com

// Iterar
foreach ($personas as $persona) {
    echo $persona["nombre"] . " (" . $persona["edad"] . " años)<br>";
}
?>

    7.3. Arrays mixtos

    
<?php
// Combinar arrays indexados y asociativos
$producto = [
    "id" => 1,
    "nombre" => "Laptop",
    "precio" => 999.99,
    "especificaciones" => [
        "procesador" => "Intel i7",
        "ram" => "16GB",
        "almacenamiento" => "512GB SSD"
    ],
    "etiquetas" => ["electrónica", "computadora", "portátil"]
];

// Acceder
echo $producto["nombre"];                      // Laptop
echo $producto["especificaciones"]["ram"];     // 16GB
echo $producto["etiquetas"][0];                // electrónica
?>

    Resumen de arrays en PHP

    • Los arrays almacenan múltiples valores en una variable
    • Arrays indexados usan números (0, 1, 2, ...)
    • Arrays asociativos usan claves personalizadas
    • foreach() es la forma más común de iterar arrays
    • count() obtiene el número de elementos
    • sort() ordena de menor a mayor
    • array_merge() combina múltiples arrays
    • Los arrays multidimensionales contienen otros arrays
