Índice

    Atajos de teclado por IDE

    Introducción

    Aquí tienes una sección por IDE con los atajos más usados para codificación. En macOS, sustituye Ctrl por ⌘ (si aplica).

    Los atajos pueden cambiar según el keymap configurado en cada IDE.

    1. NetBeans

    Top 25 shortcuts de codificación en NetBeans (Windows).

    # Atajo Acción Uso típico
    1 Ctrl + Space Autocompletar Sugerencias de métodos, clases y variables
    2 Ctrl + Shift + Space Parámetros del método Ver firma y tipos esperados
    3 Alt + Insert Generar código Constructores, getters/setters, override
    4 Ctrl + Shift + I Fix Imports Organizar/importar clases automáticamente
    5 Alt + Shift + F Reformatear código Aplicar formato según el estilo del proyecto
    6 Ctrl + / Comentar línea Activar/desactivar comentario en la línea
    7 Ctrl + Shift + / Comentar bloque Comentar/Descomentar selección
    8 Ctrl + Shift + D Duplicar línea Replicar una línea o selección
    9 Ctrl + E Eliminar línea Borrar línea completa sin seleccionar
    10 Ctrl + F Buscar Buscar texto en el archivo actual
    11 Ctrl + H Reemplazar Buscar y reemplazar en el archivo
    12 Ctrl + G Ir a línea Saltar a una línea concreta
    13 Ctrl + B Ir a declaración Abrir la definición del símbolo
    14 Alt + F7 Buscar usos Localizar referencias del símbolo
    15 Ctrl + O Ir a tipo/clase Buscar y abrir clases rápidamente
    16 Ctrl + Shift + O Ir a archivo Buscar cualquier archivo del proyecto
    17 Ctrl + Alt + O Ir a símbolo Buscar métodos o variables por nombre
    18 Ctrl + F12 Miembros del archivo Lista de métodos y campos del fichero
    19 F2 Siguiente error Navegar entre errores de compilación
    20 Ctrl + S Guardar Guardar el archivo actual
    21 Ctrl + Shift + S Guardar todo Guardar todos los archivos abiertos
    22 Ctrl + N Nuevo archivo Crear clase/archivo desde plantilla
    23 Ctrl + Shift + N Nuevo proyecto Iniciar un proyecto desde cero
    24 Ctrl + Tab Cambiar de pestaña Alternar entre archivos abiertos
    25 Ctrl + R Renombrar (refactor) Renombrar variables, métodos o clases
