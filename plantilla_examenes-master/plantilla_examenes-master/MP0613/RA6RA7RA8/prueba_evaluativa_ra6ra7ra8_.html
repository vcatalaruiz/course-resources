Enunciado

Desarrolla aplicaciones de acceso a almacenes de datos, aplicando medidas para mantener la seguridad y la integridad de la información. Utiliza correctamente objetos SQL en PHP para trabajar con diferentes drivers (MySQL, PostgreSQL,...) y hacer persistir los datos. Analiza tecnologías y mecanismos que permiten realizar esta separación y sus principales características. Entiende lo que es un ORM y sabe aplicar los conocimientos dentro de una Aplicación Web.

1. Set up (2 punto)

  • Crea un repositorio en GitHub llamado MP0613_RA6RA7RA8_EXAMEN.
  • (Requerido para evaluación) Configúralo como público.
  • Clonar el repositorio examen proporcionado en el aula virtual.
  • Instalación y ejecución del proyecto en un servidor local

2. Gestión de Issues (4 puntos cada issue)

  • Copia el Issue del repositorio examen.
  • Asígnate el Issue a ti mismo.
  • Crea una nueva rama (branch) desde GitHub.
  • Cambia a esa rama en tu entorno local.
  • Realiza las modificaciones necesarias para resolver el Issue.
  • Haz commit con el mensaje: "fix commit #numeroIssue".
  • Haz push a la rama de Github.
  • Crea un Pull Request hacia la rama principal.
  • Aprueba el PR y realiza el merge.
  • (Requerido para evaluación) Adjunta en el Issue capturas de pantalla del proyecto funcionando y aquello que demuestre que el issue ha sido resuelto.
  • Finalmente, cierra el Issue.

3. Evidencias del proceso

  • Sube al aula virtual el enlace del repositorio de github en formato .txt.
