RA1 – Introducción a Java

Objetivo: usar variables, entrada/salida y condicionales en consola.

Ejercicio

Realiza un programa RA1.java que pida por consola el nombre, la edad y la nota (0-10). El programa debe:

  • Almacenar los datos introducidos en variables.
  • Muestra el siguiente mensaje: “Hola [nombre], tienes [edad] años y tu nota es [nota].”
  • Realiza una comparación con la nota y el 5 para ver si está aprobado.
Ejemplo de salida: “Hola Juan, tienes 20 años y tu nota es 7.”
Aprobado: true

RA3 – Bucles y arrays estáticos (Convalida RA1)

Objetivo: trabajar con arrays estáticos y bucles.

Ejercicio

Desarrolla un programa RA3.java que lea 5 números enteros en un array estático (pueden ser fijos) y muestre por consola:

  • La suma de los números.
  • El número máximo y el número mínimo.
  • La cantidad de números pares e impares.

No se puede hacer con ArrayList

Se tiene que utilizar un bucle para recorrer el array

RA2 – ArrayList y métodos (Convalida RA1)

Objetivo: gestionar una lista dinámica con métodos propios.

Ejercicio

Crea un programa RA2.java que utilice un ArrayList para almacenar y gestionar una lista de colores.

  • El programa debe tener los siguientes métodos: Añadir color, Eliminar color y Mostrar colores.
  • Cada opción debe ser gestionada por un método independiente.
  • El método añadir color tiene un parámetro para recibir el color a añadir.
  • El método eliminar color pregunta al usuario que color desea eliminar.
  • El método mostrar colores no tiene parámetros, pero devuelve la lista de colores.

RA4 – Clases y objetos (Convalida RA1, RA2 y RA3)

Objetivo: modelar una clase y utilizar objetos.

Ejercicio

Diseña una clase Alumno en RA4.java con:

  • String nombre
  • int edad
  • String curso
  • ArrayList calificaciones
  • Constructor que inicialice los atributos.
  • Método mostrarMedia() que imprima la media de las calificaciones.
  • Método mostrarNotaMaxima() que imprima la nota máxima.
  • En el main, crea al menos dos objetos Alumno y muestra su información.

RA7 – Herencia, interfaces y sobrescritura (Convalida RA1, RA2, RA3 y RA4)

Objetivo: implementar herencia e interfaces en Java.

Ejercicio

A partir del ejercicio de la RA4 (si no lo has hecho hazlo), crea una clase AlumnoInternacional que herede de Alumno y haga lo siguiente:

  • Añade un nuevo atributo paisOrigen.
  • Sobrescribe el método mostrarMedia() para que también muestre el país de origen.
  • Implementa una interfaz InteresadoEnDeportes con un método mostrarDeporteFavorito(), y haz que AlumnoInternacional la implemente.
  • En el main, crea un objeto de tipo AlumnoInternacional y muestra su información, incluyendo la media, el país de origen y su deporte favorito.

RA5 – Lógica Swing (Convalida RA1, RA2, RA3 y RA4)

Objetivo: programar el comportamiento de un formulario Swing.

Ejercicio

Mediante JSwing, haz el ejercico de la RA4 pero con una interfaz gráfica. Para ello, crea un proyecto Java que cumpla con la estructura MVC

  • El modelo es la clase Alumno con sus atributos y métodos.
  • La vista está formada por la pantalla Home y tres pantallas adicionales (accesibles desde Home): Registro, Listado y Detalle.
  • El controlador gestiona la lógica de la aplicación, como el registro de alumnos, la navegación entre pantallas y la actualización de la vista.

RA6 – HashMap (Convalida RA1, RA2 y RA3)

Objetivo: almacenar y buscar datos con HashMap.

Ejercicio

Crea un programa RA6.java que implemente un HashMap< String, String> para almacenar los vehículos de un concesionario:

  • El programa empieza pidiendo al usuario la matrícula y la marca de 5 vehículos mediante un bucle.
  • Almacena cada par matrícula-marca en el HashMap, utilizando la matrícula como clave.
  • Una vez introducidos los datos, se llama al método mostrarVehiculo(String matrícula) para que devuelva la marca del vehículo o un mensaje de error si no se encuentra.
  • Muestra el resultado de la búsqueda por consola.

RA8 y RA9 – Bases de datos (Convalida RA1, RA2, RA3, RA4)

Objetivo: diseñar la tabla y consultas SQL para una base de datos relacional de alumnos.

Ejercicio

Haz el ejercicio de la RA4 implementando una BBDD relacional.

  • Diseña la tabla SQL para almacenar la información de los alumnos, incluyendo su nombre, edad, curso y calificaciones.
  • Escribe un método con una consulta SQL para insertar un nuevo alumno en la tabla.
  • Escribe un método con una consulta SQL para actualizar la edad de un alumno específico.
  • Escribe un método con una consulta SQL para eliminar un alumno de la tabla.
  • Escribe un método con una consulta SQL para recuperar todos los alumnos que están en un curso específico.
