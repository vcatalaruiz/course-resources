Enunciado

Crea un programa para simular la gestión de un concesionario de vehículos. La información se almacenará en un fichero de texto, simulando una base de datos interna. El entorno gráfico debe implementarse con Swing, utilizando botones, etiquetas y cajas de texto para mostrar la información e interactuar con el usuario.

3. Interfaz Gráfica (Swing) (5 puntos)

  • Implementar una ventana principal con JFrame, submenus con JDialog y paneles de información JOptionPane.
  • Insertar vehículo: Preguntar si es un coche o una moto y solicitar sus datos. [1 punto]
  • Leer vehículo: Pedir la matrícula para localizarlo y mostrar sus detalles. [1 punto]
  • Eliminar vehículo: Pedir la matrícula para localizarlo y eliminar. [1 punto]
  • Listar coches: Mostrar todos los coches del catálogo. [1 punto]
  • Listar motos: Mostrar todas las motos del catálogo. [1 punto]

4. Persistencia de Datos (2 puntos)

  • Guardar los datos al cerrar el programa
  • Cargar los datos al iniciar

5. Validaciones y Restricciones (1 punto)

  • Matrícula única: No puede haber dos vehículos con la misma matrícula.
  • Tratamiento de errores: Mostrar un diálogo asociado a una excepción personalizada.
