Enunciado

Crea un programa para simular la gestión de un hotel. La información se almacenará en un fichero de texto, simulando una base de datos interna. El entorno gráfico debe implementarse con Swing, utilizando botones, etiquetas y cajas de texto para mostrar la información e interactuar con el usuario.

3. Interfaz Gráfica (Swing) (5 puntos)

  • Implementar una ventana principal con JFrame, submenus con JDialog y paneles de información JOptionPane.
  • Registrar Habitacion: Añadir una Habitacion con sus datos. Disponible por defecto. [1 punto]
  • Buscar Habitacion: Pedir el número para localizarla y mostrar sus detalles. [1 punto]
  • Eliminar Habitacion: Pedir el número para localizarla y eliminar. [1 punto]
  • Cambiar disponibilidad: Reservar o liberar una Habitacion por su número. [1 punto]
  • Listar habitaciones: Mostrar las habitaciones. [1 punto]

4. Persistencia de Datos (2 puntos)

  • Guardar los datos al cerrar el programa
  • Cargar los datos al iniciar

5. Validaciones y Restricciones (1 punto)

  • Número único: No puede haber dos habitaciones con el mismo número.
  • Tratamiento de errores: Mostrar un diálogo asociado a una excepción personalizada.
