Enunciado

Crea un programa para simular la gestión de una biblioteca digital. La información se almacenará en un fichero de texto, simulando una base de datos interna. El entorno gráfico debe implementarse con Swing, utilizando botones, etiquetas y cajas de texto para mostrar la información e interactuar con el usuario.

3. Interfaz Gráfica (Swing) (5 puntos)

  • Implementar una ventana principal con JFrame, submenus con JDialog y paneles de información JOptionPane.
  • Añadir documento: Preguntar si es un libro o una revista y solicitar sus datos. [1 punto]
  • Buscar documento: Pedir el ISBN para localizarlo y mostrar sus detalles. [1 punto]
  • Listar libros: Mostrar solo los libros de la biblioteca. [1 punto]
  • Listar revistas: Mostrar solo las revistas de la biblioteca. [1 punto]
  • Contar documentos: Mostrar número total de documentos. [1 punto]

4. Persistencia de Datos (2 puntos)

  • Guardar los datos al cerrar el programa
  • Cargar los datos al iniciar

5. Validaciones y Restricciones (1 punto)

  • ISBN único: No puede haber dos documentos con el mismo ISBN.
  • Tratamiento de errores: Mostrar un diálogo asociado a una excepción personalizada.
