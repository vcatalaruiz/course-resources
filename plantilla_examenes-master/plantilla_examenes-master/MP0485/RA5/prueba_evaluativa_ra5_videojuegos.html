Enunciado

Crea un programa para simular la gestión de una tienda de videojuegos. La información se almacenará en un fichero de texto, simulando una base de datos interna. El entorno gráfico debe implementarse con Swing, utilizando botones, etiquetas y cajas de texto para mostrar la información e interactuar con el usuario.

3. Interfaz Gráfica (Swing) (5 puntos)

  • Diseño: Implementar las ventanas con jFrame y diálogos de información JOptionPane cuando sean necesarios.
  • Añadir videojuego: Solicitar sus datos y validarlos. [1 punto]
  • Buscar videojuego: Pedir el código para localizarlo y mostrar sus detalles. [1 punto]
  • Eliminar videojuego: Eliminar un videojuego por su código. [1 punto]
  • Listar videojuegos : Mostrar todos los videojuegos. [1 punto]

4. Persistencia de Datos (2 puntos)

  • Guardar los datos al cerrar el programa
  • Cargar los datos al iniciar

5. Validaciones y Restricciones (1 punto)

  • Código único: No puede haber dos Videojuegoss con el mismo código.
  • Tratamiento de errores: Mostrar un diálogo asociado a una excepción personalizada.
