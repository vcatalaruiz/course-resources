Enunciado

Crea un programa para simular la gestión de Polizas de seguros y siniestros. La información se almacenará en un fichero de texto, simulando una base de datos interna. El entorno gráfico debe implementarse con Swing, utilizando botones, etiquetas y cajas de texto para mostrar la información e interactuar con el usuario.

3. Interfaz Gráfica (Swing) (5 puntos)

  • Implementar una ventana principal con JFrame, submenus con JDialog y paneles de información JOptionPane.
  • Registrar Poliza: Añadir una Poliza con sus datos (número, cliente, tipo, prima). [1 punto]
  • Buscar Poliza: Pedir el número para localizarla y mostrar sus detalles. [1 punto]
  • Eliminar Poliza: Eliminar una Poliza por su número. [1 punto]
  • Actualizar Poliza: Modificar el estado de una Poliza (activa/inactiva). [1 punto]
  • Listar Polizas: Mostrar todas las Polizas. [1 punto]

4. Persistencia de Datos (2 puntos)

  • Guardar los datos al cerrar el programa
  • Cargar los datos al iniciar

5. Validaciones y Restricciones (1 punto)

  • Número único: No puede haber dos Polizas con el mismo número.
  • Tratamiento de errores: Mostrar un diálogo asociado a una excepción personalizada.
