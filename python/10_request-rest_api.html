    Request a una API REST en Python

    Una API REST permite a una aplicación comunicarse con otra mediante peticiones HTTP. En Python, la forma más habitual de realizar estas peticiones es utilizando la librería requests, que simplifica el trabajo con los métodos HTTP más comunes.

    • Comunicación cliente-servidor
    • Uso de métodos HTTP (GET, POST, PUT, DELETE)
    • Intercambio de datos en formato JSON

    Petición GET

    El método GET se utiliza para solicitar información a un recurso de la API. No modifica datos en el servidor y suele devolver la respuesta en formato JSON.

    Ejemplo 
    import requests

url = "https://api.ejemplo.com/usuarios"
respuesta = requests.get(url)

if respuesta.status_code == 200:
    datos = respuesta.json()
    print(datos)
else:
    print("Error en la petición")

    Petición POST

    El método POST se emplea para enviar datos al servidor, normalmente para crear nuevos recursos. Los datos se envían en el cuerpo de la petición.

    Ejemplo 
    import requests

url = "https://api.ejemplo.com/usuarios"
nuevo_usuario = {
    "nombre": "Ana",
    "email": "ana@ejemplo.com"
}

respuesta = requests.post(url, json=nuevo_usuario)

if respuesta.status_code == 201:
    print("Usuario creado correctamente")
else:
    print("Error al crear el usuario")

    Gestión de errores y respuestas

    Es importante comprobar el código de estado HTTP devuelto por la API para saber si la petición se ha realizado correctamente o si se ha producido un error.

    • 200–299: petición correcta
    • 400–499: error del cliente
    • 500–599: error del servidor
    Ejemplo 
    try:
    respuesta = requests.get(url)
    respuesta.raise_for_status()
    print(respuesta.json())
except requests.exceptions.RequestException as e:
    print("Error:", e)
