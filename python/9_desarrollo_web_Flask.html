    Desarrollo Web con Flask

    ¿Qué es una aplicación web?

    Una aplicación web es un software que se ejecuta en un servidor y se accede a través de un navegador web. No necesita instalación en el equipo del usuario y funciona mediante internet o una red local.

    • Se accede mediante un navegador (Chrome, Firefox, Edge…)
    • El usuario interactúa mediante páginas web
    • El procesamiento principal ocurre en el servidor

    Web estática vs web dinámica

    Una web estática muestra siempre el mismo contenido, mientras que una web dinámica genera contenido diferente según el usuario o la información recibida.

    • Estática: HTML y CSS sin lógica (ej. una página informativa)
    • Dinámica: Contenido generado por el servidor (ej. formularios, usuarios)

    Modelo cliente–servidor

    El modelo cliente–servidor separa responsabilidades: el cliente solicita información y el servidor la procesa y responde.

    • Cliente: navegador web
    • Servidor: aplicación web (Flask)
    • Comunicación mediante peticiones HTTP

    Rol del backend

    El backend se encarga de la lógica de la aplicación: procesa datos, toma decisiones y genera respuestas.

    • Gestión de rutas
    • Procesamiento de formularios
    • Generación de contenido dinámico

    ¿Qué es Flask?

    Flask es un framework web ligero escrito en Python que permite crear aplicaciones web de forma sencilla y flexible.

    Características:

    • Framework minimalista
    • Basado en Python
    • Ideal para aprender desarrollo web

    Características principales de Flask

    • Ligero y minimalista
    • Fácil de aprender
    • Extensible mediante librerías
    • Control total sobre la estructura

    Casos de uso de Flask

    Flask se utiliza tanto para proyectos pequeños como para aplicaciones más completas. Suele ser la opción preferida para:

    • Aplicaciones educativas
    • APIs sencillas
    • Prototipos rápidos

    Ventajas frente a otros frameworks

    • Menor complejidad inicial
    • No impone una estructura rígida
    • Ideal para aprender backend

    Estructura básica de un proyecto Flask

    Un proyecto Flask se organiza en archivos y carpetas para separar lógica, plantillas y recursos.

    • app.py: archivo principal
    • templates/: HTML
    • static/: CSS, imágenes

    Tutorial

    En este tutorial aprenderás a crear una aplicación web sencilla con Flask que tendrá dos pantallas: una página principal y una página de saludo personalizada. El objetivo es entender el flujo completo de una aplicación web backend con Python.

    • Crear una aplicación Flask desde cero
    • Trabajar con rutas
    • Usar plantillas HTML
    • Mostrar información dinámica

    Requisitos previos

    Antes de comenzar, es necesario tener preparado el entorno de trabajo.

    • Python instalado (versión 3.x)
    • Editor de código (VS Code recomendado)
    • Conocimientos básicos de Python

    Instalación de Flask

    Flask se instala fácilmente usando el gestor de paquetes de Python. 

    pip install flask

    Estructura del proyecto

    Organizaremos el proyecto separando la lógica Python de las páginas HTML. 

    proyecto_flask/
│
├── app.py
└── templates/
    ├── index.html
    └── saludo.html

    Archivo principal app.py

    Este archivo contiene la lógica principal de la aplicación Flask y define las rutas. 

    from flask import Flask, render_template, request

app = Flask(__name__)

@app.route('/')
def inicio():
    return render_template('index.html')

@app.route('/saludo', methods=['POST'])
def saludo():
    nombre = request.form['nombre']
    return render_template('saludo.html', nombre=nombre)

if __name__ == '__main__':
    app.run(debug=True)

    Pantalla 1: Página principal

    La primera pantalla muestra un formulario donde el usuario introduce su nombre.

    Archivo index.html

    Este HTML contiene el formulario que envía datos al servidor. 

    <!DOCTYPE html>
<html>
<head>
    <title>Inicio</title>
</head>
<body>
    <h1>Bienvenido</h1>
    <form action="/saludo" method="post">
        <input type="text" name="nombre" placeholder="Introduce tu nombre">
        <button type="submit">Enviar</button>
    </form>
</body>
</html>

    Pantalla 2: Página de saludo

    La segunda pantalla muestra un mensaje personalizado utilizando el dato introducido por el usuario.

    Archivo saludo.html

    Flask envía el nombre a la plantilla y esta lo muestra dinámicamente. 

    <!DOCTYPE html>
<html>
<head>
    <title>Saludo</title>
</head>
<body>
    <h1>Hola {{ nombre }}</h1>
    <p>Gracias por usar nuestra web con Flask.</p>
    <a href="/">Volver</a>
</body>
</html>

    Flujo de la aplicación

    El funcionamiento completo de la web sigue estos pasos:

    • El usuario accede a la página principal
    • Introduce su nombre en el formulario
    • El servidor recibe el dato
    • Se genera una respuesta personalizada

    Ejecución de la aplicación

    Para arrancar la web, ejecuta el archivo principal y accede desde el navegador. 

    python app.py

    Accede a: http://127.0.0.1:5000

    Ejercicio práctico:

    Crea una aplicación web con Flask que realice lo siguiente:

    • Mostrar una página de inicio con un formulario para introducir el nombre de un producto y su precio.
    • Al enviar el formulario, mostrar una página de resumen con el nombre del producto, su precio.
    • Indicar si el producto es caro (precio > 100) o barato (precio <= 100).
    • Incluir CSS para mejorar la apariencia de la aplicación.
