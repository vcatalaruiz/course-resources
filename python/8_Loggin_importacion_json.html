    Ficheros JSON en Python

    JSON es un formato de texto muy utilizado para almacenar y compartir datos estructurados. Python dispone del módulo estándar json para convertir diccionarios y listas a JSON y viceversa. Es habitual usarlo para ficheros de configuración o datos sencillos.

    Explicación

    En este ejemplo se crea un diccionario con datos de configuración, se guarda en un fichero JSON y posteriormente se vuelve a cargar para acceder a sus valores.

    Ejemplo 
    import json  # Importamos el módulo estándar json

# Diccionario con datos de configuración
configuracion = {
    "app": "MiAplicacion",
    "version": 1,
    "modo": "produccion",
    "debug": False
}

# Escritura del diccionario en un fichero JSON
with open("config.json", "w", encoding="utf-8") as f:
    json.dump(configuracion, f, indent=4, ensure_ascii=False)

# Lectura del fichero JSON y conversión a diccionario Python
with open("config.json", "r", encoding="utf-8") as f:
    datos = json.load(f)

# Acceso a un valor concreto
print("Modo:", datos["modo"])

    JSON: control de errores y validación básica

    Al trabajar con ficheros externos es importante controlar posibles errores: el fichero puede no existir, estar mal formado o no contener las claves necesarias. Por ello se recomienda usar excepciones y validaciones simples antes de utilizar los datos.

    Explicación

    La función intenta cargar un fichero JSON y comprueba que exista una clave obligatoria. Si ocurre algún error, se muestra un mensaje adecuado.

    Ejemplo 
    import json

def cargar_config(ruta):
    try:
        # Intentamos abrir y leer el fichero JSON
        with open(ruta, "r", encoding="utf-8") as f:
            config = json.load(f)

        # Validación básica de la estructura
        if "modo" not in config:
            raise ValueError("Falta la clave 'modo'")

        return config

    except FileNotFoundError:
        print("El fichero no existe")
    except json.JSONDecodeError:
        print("El fichero no es un JSON válido")
    except ValueError as e:
        print("Error de configuración:", e)

cargar_config("config.json")

    JSON: Ejemplo de recorrido de un array dentro de un JSON

    En este ejemplo se muestra cómo recorrer un array de objetos dentro de un JSON para acceder a sus valores. Por ejemplo, una lista de tareas con una etiqueta "completada" que indica si la tarea está hecha o no.

    Explicación

    Se carga un JSON con una lista de tareas y se recorre para mostrar solo las tareas no completadas.

    Ejemplo de JSON de tareas

    {
    "tareas": [
        {"nombre": "Comprar leche", "completada": False},
        {"nombre": "Lavar el coche", "completada": True},
        {"nombre": "Hacer ejercicio", "completada": False}
    ]
}
    Ejemplo 
    import json

# Supongamos que tenemos un JSON con una lista de tareas
with open("tareas.json", "r", encoding="utf-8") as f:
    data = json.load(f)

# Recorremos la lista de tareas y mostramos solo las no completadas
for tarea in data["tareas"]:
    if not tarea["completada"]:
        print(f"Tarea: {tarea['nombre']}")

    Logging en Python

    El módulo logging permite registrar información sobre la ejecución del programa. Es una alternativa profesional a print(), ya que permite clasificar los mensajes por nivel y controlar su formato y destino.

    Explicación

    Se configura el sistema de logging para mostrar mensajes informativos y se registran distintos tipos de mensajes según su importancia.

    Ejemplo 
    import logging

# Configuración básica del sistema de logging
logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s | %(levelname)s | %(message)s"
)

# Registro de distintos mensajes
logging.info("Aplicación iniciada correctamente")
logging.warning("Falta un valor opcional, se usará uno por defecto")
logging.error("Se ha producido un error en la operación")

    Logging en ficheros

    En aplicaciones reales es muy habitual guardar los logs en un fichero para poder revisarlos más tarde. Esto permite analizar errores y el comportamiento del programa tras su ejecución.

    Explicación

    En este ejemplo los mensajes se guardan en un fichero app.log y se registra automáticamente el error completo cuando ocurre una excepción.

    Ejemplo 
    import logging

# Configuración del logging para escribir en un fichero
logging.basicConfig(
    filename="app.log",
    level=logging.INFO,
    format="%(asctime)s | %(levelname)s | %(message)s"
)

logger = logging.getLogger(__name__)

def dividir(a, b):
    try:
        # Operación que puede provocar un error
        return a / b
    except Exception:
        # Registro del error con información completa
        logger.exception("Error al dividir %s entre %s", a, b)
        return None

dividir(10, 0)

    Librerías de terceros

    Python permite ampliar sus funcionalidades mediante librerías externas que resuelven tareas comunes. Estas librerías se instalan con pip y se importan como cualquier otro módulo. Su uso ahorra tiempo y simplifica el desarrollo.

    Instalación de una librería:

    pip install nombre_de_la_librería

    Algunas librerías populares:

    • requests: consumo de APIs y peticiones HTTP.
    • python-dotenv: gestión de variables de entorno. Ejemplo: Microsoft Azure .env
    • rich: salida por consola más clara y legible.
    • tabulate: presentación de datos en tablas de texto.
    • click: creación de programas de línea de comandos.
    • schedule: ejecución de tareas periódicas.
    Explicación

    A continuación se muestran ejemplos sencillos de uso de algunas librerías habituales en tareas cotidianas como peticiones web, configuración y presentación de datos.

    Ejemplo 
    # Petición HTTP sencilla con requests
import requests

respuesta = requests.get("https://swapi.dev/api/people/1/")
print(respuesta.name)  # Imprime el nombre del personaje

# Carga de variables de entorno desde un fichero .env
from dotenv import load_dotenv
import os
# Cargar variables de entorno desde el fichero .env
# Acceso al fichero en el directorio actual
fichero_env = os.path.join(os.getcwd(), ".env")
load_dotenv(fichero_env)
clave_api = os.getenv("CLAVE_API")
print("Clave API:", clave_api)

# Mostrar datos en forma de tabla por consola
from tabulate import tabulate

datos = [["Ana", 8.5], ["Luis", 6.0]]
print(tabulate(datos, headers=["Nombre", "Nota"]))
