    Laravel Livewire

    1. ¿Qué es Livewire?

    Livewire es un framework full-stack para Laravel que permite crear interfaces dinámicas sin escribir JavaScript. Funciona combinando componentes PHP con vistas Blade reactivas.

    2. Instalación

    2.1. Instalar paquete

    
composer require livewire/livewire

    2.2. Incluir assets

    
<!-- layout.blade.php -->
@livewireStyles
...
@livewireScripts

    3. Componentes Livewire

    3.1. Crear componente

    
php artisan make:livewire contador

    3.2. Clase del componente

    
namespace App\Livewire;

use Livewire\Component;

class Contador extends Component
{
    public int $count = 0;

    public function increment(): void
    {
        $this->count++;
    }

    public function render()
    {
        return view('livewire.contador');
    }
}

    3.3. Vista del componente

    
<div>
    <h4>{{ $count }}</h4>
    <button wire:click="increment">Sumar</button>
</div>

    4. Data Binding

    4.1. Enlace bidireccional

    
<input type="text" wire:model="nombre">
<p>Hola, {{ $nombre }}</p>

    4.2. Propiedad en el componente

    
public string $nombre = '';

    5. Eventos y métodos

    5.1. Llamar métodos

    
<button wire:click="guardar">Guardar</button>

    5.2. Método en componente

    
public function guardar(): void
{
    // Lógica de guardado
}

    6. Validación

    6.1. Reglas

    
protected array $rules = [
    'email' => 'required|email'
];

    6.2. Validar y mostrar errores

    
public function guardar(): void
{
    $this->validate();
    // Guardar datos
}

    
<input type="text" wire:model="email">

    8. Paginación

    8.1. Usar WithPagination

    
use Livewire\WithPagination;

class ListaUsuarios extends Component
{
    use WithPagination;

    public function render()
    {
        return view('livewire.lista-usuarios', [
            'usuarios' => User::paginate(10)
        ]);
    }
}

    8.2. Mostrar paginación

    
{{ $usuarios->links() }}
